# Conceito de Design: Terminal Interativo de Desenvolvedor

## Visão Geral

O portfólio será construído com a estética de um **terminal de linha de comando** moderno, combinando a familiaridade de um ambiente de desenvolvimento com alta interatividade e design contemporâneo. Este conceito é único, demonstra habilidades técnicas diretamente através do design e cria uma experiência memorável para recrutadores.

## Estilo Visual

**Paleta de Cores:**
- Fundo: Preto profundo (#0a0a0a) com leve gradiente
- Texto principal: Verde neon (#00ff41) para simular terminal clássico
- Texto secundário: Cinza claro (#a0a0a0)
- Destaques: Ciano (#00d9ff), Magenta (#ff00ff), Amarelo (#ffff00)
- Bordas: Verde escuro com brilho sutil

**Tipografia:**
- Fonte principal: JetBrains Mono (monoespaçada, moderna)
- Fallback: Fira Code, Consolas, monospace

**Elementos Visuais:**
- Efeito de scanline sutil no fundo
- Cursor piscante animado
- Efeito de digitação (typing effect) para textos
- Brilho (glow) nos elementos interativos
- Partículas ou grid no fundo para profundidade

## Estrutura de Navegação

A navegação será feita através de um sistema de "comandos" visíveis como botões estilizados:

1. **Prompt Inicial:** Tela de boas-vindas com ASCII art e prompt de comando
2. **Comandos Disponíveis:**
   - `whoami` → Seção "Sobre Mim"
   - `ls projects` → Seção "Projetos"
   - `cat skills.txt` → Seção "Habilidades"
   - `contact --help` → Seção "Contato"
   - `clear` → Voltar ao início

## Seções do Portfólio

### 1. Tela Inicial (Prompt)
- ASCII art com nome do desenvolvedor
- Mensagem de boas-vindas com efeito de digitação
- Lista de comandos disponíveis
- Cursor piscante

### 2. Sobre Mim (`whoami`)
- Bio com efeito de digitação
- Foto de perfil com borda estilo terminal
- Informações pessoais formatadas como output de comando
- Links para redes sociais (GitHub, LinkedIn, etc.)

### 3. Projetos (`ls projects`)
- Cards de projetos com estilo de "arquivo" de terminal
- Hover effects com brilho verde
- Informações: nome, descrição, tecnologias, links (GitHub, demo)
- Filtros por tecnologia

### 4. Habilidades (`cat skills.txt`)
- Visualização de tecnologias como "logs" de sistema
- Barras de progresso estilizadas como loading bars de terminal
- Categorias: Frontend, Backend, Ferramentas, Soft Skills

### 5. Contato (`contact --help`)
- Formulário de contato estilizado como input de terminal
- Links para redes sociais
- Email e outras informações de contato

## Interatividade

- **Animações de transição:** Efeito de "executar comando" ao navegar
- **Efeito de digitação:** Textos aparecem como se fossem digitados
- **Cursor animado:** Cursor piscante em elementos de input
- **Hover states:** Brilho e mudança de cor em elementos interativos
- **Responsividade:** Layout adaptável para mobile (terminal simplificado)

## Diferencial

Este portfólio se destaca por:
1. **Demonstrar habilidades técnicas:** O design em si mostra domínio de CSS, animações e JavaScript
2. **Criar experiência memorável:** Recrutadores vão lembrar do "portfólio terminal"
3. **Ser funcional e bonito:** Combina estética única com usabilidade
4. **Refletir a cultura dev:** Usa elementos familiares para desenvolvedores (terminal, comandos)
